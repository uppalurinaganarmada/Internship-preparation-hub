🐍 Python Interview Questions

What is the difference between List and Tuple?

List is mutable (can be modified). Tuple is immutable (cannot be modified). Lists use [] while tuples use ().

What is a Dictionary?

A Dictionary stores data in key-value pairs. Example: {'name':'Narmada','age':21}

What is Exception Handling?

Exception handling helps manage runtime errors using try, except, else and finally blocks.

What are Functions?

Functions are reusable blocks of code that perform specific tasks and improve code reusability.

What is OOP?

Object Oriented Programming is a programming paradigm based on classes and objects. Concepts include inheritance, polymorphism, encapsulation and abstraction.

🗄️ SQL Interview Questions

What is SQL?

SQL stands for Structured Query Language. It is used to manage and manipulate relational databases.

Difference between DELETE, DROP and TRUNCATE?

DELETE removes selected rows.
TRUNCATE removes all rows quickly.
DROP removes the entire table structure.

What are Joins?

Joins combine data from multiple tables. Examples include INNER JOIN, LEFT JOIN, RIGHT JOIN and FULL JOIN.

What is a Primary Key?

A Primary Key uniquely identifies each record in a table. It cannot contain NULL values.

What is a Foreign Key?

A Foreign Key creates a relationship between two tables. It references the Primary Key of another table.

📄 Resume Tips

  • Keep your resume concise and professional.
  • Add relevant projects and certifications.
  • Use action verbs like Developed, Created, Designed.
  • Highlight technical and soft skills.
  • Customize your resume for each job role.
  • Proofread before submitting.

🎤 HR Interview Questions

Tell me about yourself.

I am a motivated graduate with skills in Python, SQL, HTML and CSS. I enjoy building projects and continuously learning new technologies.

Why should we hire you?

I am a quick learner, hardworking and eager to contribute to the organization while improving my technical and professional skills.

What are your strengths?

My strengths include problem-solving, adaptability, teamwork and continuous learning.

What are your weaknesses?

I sometimes spend extra time perfecting my work, but I am learning to balance quality and deadlines.

Where do you see yourself in 5 years?

I see myself as a skilled professional contributing to impactful projects and taking on greater responsibilities within the organization.
